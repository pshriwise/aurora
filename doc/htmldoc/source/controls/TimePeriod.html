      TimePeriod

      The TimePeriod class derives from ConditionalEnableControl.

      TimePeriod objects are designed to control the active state of a MOOSE object, based on whether the time value is in some range.

      For example, consider a simulation that contains two "diff0" and "diff1". Initially, "diff0" is active and after time 0.49 "diff1" becomes active and "diff0" is disabled. The following code snippet demonstrates how this switching of kernels is achieved with the TimePeriod object.

      [Controls]
  [./diff]
    type = TimePeriod
    enable_objects = 'Kernel::diff0'
    disable_objects = '*::diff1'
    start_time = '0'
    end_time = '0.49'
  [../]
[]
