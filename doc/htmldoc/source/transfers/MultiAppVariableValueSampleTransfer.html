      MultiAppVariableValueSampleTransfer

      Transfers the value of a variable within the master application at each sub-application position and transfers the value to a field variable on the sub-application(s).

      Example Input File Syntax

      The following input file snippet demonstrates the use of the MultiAppVariableValueSampleTransfer to transfer the value of a field variable at the sub-application positions to a field variable on each sub-application.

      [Transfers]
  [sample_transfer]
    source_variable = u
    direction = to_multiapp
    variable = from_master
    type = MultiAppVariableValueSampleTransfer
    multi_app = sub
  []
[]
