      AxisymmetricCenterlineAverageValue

      Computes the average value of a variable on a sideset located along the centerline of an axisymmetric model.

      Description

      'AxisymmetricCenterlineAverageValue' is a specialized case of the SideAverageValue postprocessor for cases in which the sideset on which the average value is being calculated is along the centerline axis in axisymmetric simulations.

      Example Input syntax

      [Postprocessors]
  [./average]
    type = AxisymmetricCenterlineAverageValue
    boundary = left
    variable = u
  [../]
[]
