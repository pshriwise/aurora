      VariableResidual

      Computes the L2 norm of the residual of a single variable in the solution vector.

      This postprocessor allows a user to retrieve the residual at different times in the simulation using the execute_on parameter.

      The residual for the entire system may be found using the Residual postprocessor.

      Example input syntax

      In this example, the residual is output for variable u and v with two VariableResidual postprocessors.

      [Postprocessors]
  [./u_res_l2]
    type = VariableResidual
    variable = u
  [../]

  [./v_res_l2]
    type = VariableResidual
    variable = v
  [../]
[]
