      ArrayHFEMDirichletBC

      Description

      This boundary condition applies Dirichlet boundary conditions with HFEM (hybrid finite element method) to all components of an array variable. Its standard variable version is HFEMDirichletBC. Different boundary values for components can be assigned.

      Example Input Syntax

        [all]
    type = ArrayHFEMDirichletBC
    boundary = 'left right top bottom'
    variable = u
    lowerd_variable = lambdab
  []
      (../../../SoftwareDownloads/moose/test/tests/kernels/hfem/array_dirichlet.i)
